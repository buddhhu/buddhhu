SendVoteRequest

Only users can use this method. See code examples.

Known RPC errors

This request can cause 2 known errors:

MessagePollClosedErrorThe poll was closed and can no longer be voted on.
OptionInvalidErrorThe option specified is invalid and does not exist in the target poll.

You can import these from telethon.errors.

Example

from telethon.sync import TelegramClient
from telethon import functions, types

with TelegramClient(name, api_id, api_hash) as client:
    result = client(functions.messages.SendVoteRequest(
        peer='username',
        msg_id=42,
        options=[b'arbitrary\x7f data \xfa here']
    ))
    print(result.stringify())
