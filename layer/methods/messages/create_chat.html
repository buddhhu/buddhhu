CreateChatRequest

Only users can use this method. See code examples.

---functions---

Parameters

usersInputUserAnything entity-like will work if the library can find its Input version (e.g., usernames, Peer, User or Channel objects, etc.). A list must be supplied.
titlestring

Known RPC errors

This request can cause 2 known errors:

UsersTooFewErrorNot enough users (to create a chat, for example).
UserRestrictedErrorYou're spamreported, you can't create channels or chats..

You can import these from telethon.errors.

Example

from telethon.sync import TelegramClient
from telethon import functions, types

with TelegramClient(name, api_id, api_hash) as client:
    result = client(functions.messages.CreateChatRequest(
        users=['username'],
        title='My awesome title'
    ))
    print(result.stringify())
