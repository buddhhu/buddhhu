GetAttachedStickersRequest

Only users can use this method. See code examples.

---functions---
messages.getAttachedStickers#cc5b67cc media:InputStickeredMedia = Vector<StickerSetCovered>

Returns

A list of the following type is returned.

StickerSetCovered

This type can be an instance of either:

StickerSetCoveredStickerSetFullCovered
StickerSetMultiCovered

Parameters

mediaInputStickeredMedia

Known RPC errors

This request can't cause any RPC error as far as we know.

Example

from telethon.sync import TelegramClient
from telethon import functions, types

with TelegramClient(name, api_id, api_hash) as client:
    result = client(functions.messages.GetAttachedStickersRequest(
        media=types.InputStickeredMediaPhoto(
            id=types.InputPhoto(
                id=-12398745604826,
                access_hash=-12398745604826,
                file_reference=b'arbitrary\x7f data \xfa here'
            )
        )
    ))
    for x in result:
        print(x)
