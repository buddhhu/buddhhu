Known RPC errors

This request can cause 4 known errors:

BroadcastIdInvalidErrorThe channel is invalid.
LinkNotModifiedErrorThe channel is already linked to this group.
MegagroupIdInvalidErrorThe group is invalid.
MegagroupPrehistoryHiddenErrorYou can't set this discussion group because it's history is hidden.

You can import these from telethon.errors.
