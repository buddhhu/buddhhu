SetSecureValueErrorsRequest

Only bots can use this method. See code examples.

---functions---
users.setSecureValueErrors#90c894b5 id:InputUser errors:Vector<SecureValueError> = Bool

Returns

Bool

This type has no instances available.

Parameters

idInputUserAnything entity-like will work if the library can find its Input version (e.g., usernames, Peer, User or Channel objects, etc.).
errorsSecureValueErrorA list must be supplied.

Known RPC errors

This request can't cause any RPC error as far as we know.

Example

from telethon.sync import TelegramClient
from telethon import functions, types

with TelegramClient(name, api_id, api_hash) as client:
    result = client(functions.users.SetSecureValueErrorsRequest(
        id='username',
        errors=[types.SecureValueErrorData(
            type=types.SecureValueTypePersonalDetails(),
            data_hash=b'arbitrary\x7f data \xfa here',
            field='some string here',
            text='some string here'
        )]
    ))
    print(result)
