GetMultiWallPapersRequest

Only users can use this method. See code examples.

---functions---
account.getMultiWallPapers#65ad71dc wallpapers:Vector<InputWallPaper> = Vector<WallPaper>

Returns

A list of the following type is returned.

WallPaper

This type can be an instance of either:

WallPaperWallPaperNoFile

Parameters

wallpapersInputWallPaperA list must be supplied.

Known RPC errors

This request can't cause any RPC error as far as we know.

Example

from telethon.sync import TelegramClient
from telethon import functions, types

with TelegramClient(name, api_id, api_hash) as client:
    result = client(functions.account.GetMultiWallPapersRequest(
        wallpapers=[types.InputWallPaper(
            id=-12398745604826,
            access_hash=-12398745604826
        )]
    ))
    for x in result:
        print(x)
