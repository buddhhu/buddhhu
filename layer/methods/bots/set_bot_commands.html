SetBotCommandsRequest

Only bots can use this method. See code examples.

---functions---

Known RPC errors

This request can cause 3 known errors:

BotCommandDescriptionInvalidErrorThe command description was empty, too long or had invalid characters used.
BotCommandInvalidError.
LangCodeInvalidError.

You can import these from telethon.errors.

Example

from telethon.sync import TelegramClient
from telethon import functions, types

with TelegramClient(name, api_id, api_hash) as client:
    result = client(functions.bots.SetBotCommandsRequest(
        scope=types.BotCommandScopeDefault(),
        lang_code='en',
        commands=[types.BotCommand(
            command='some string here',
            description='some string here'
        )]
    ))
    print(result)
